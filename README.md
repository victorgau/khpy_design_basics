# 美工設計基礎觀念

# 設計元素

要有一個賞心悅目的設計，需要好好思考以下的設計元素：

* Typography
* Color
* Layout
* Images
* Icons
* Logo

## Typography (字型)

不同的字體對不同的受眾有不同的涵義跟感受。

* Serif
* San Serif
* Script
* 楷書
* 明體
* 黑體
* 卡通體

## Color (顏色)

不同的顏色會給人不同的感覺。

* red
* green
* blue
* gray
* orange
* yellow

## Layout (佈局)

* 版面應該使用什麼樣的 Layout？
* 單一個欄位？或多個欄位？
* 如果是網頁，怎麼做響應式設計？
